182 /* Scan the string starting at ptr until the end of the next complete
183  token, but do not scan past eptr. Return an integer giving the
184  type of token.
185 
186  Return XML_TOK_NONE when ptr == eptr; nextTokPtr will not be set.
187 
188  Return XML_TOK_PARTIAL when the string does not contain a complete
189  token; nextTokPtr will not be set.
190 
191  Return XML_TOK_INVALID when the string does not start a valid
192  token; nextTokPtr will be set to point to the character which made
193  the token invalid.
194 
195  Otherwise the string starts with a valid token; nextTokPtr will be
196  set to point to the character following the end of that token.
197 
198  Each data character counts as a single token, but adjacent data
199  characters may be returned together. Similarly for characters in
200  the prolog outside literals, comments and processing instructions.
201 */
